# scripts

小脚本，爱生活。

- [poems](https://github.com/yeshimei/ntbl-scripts/tree/master/scripts/poems) - 列出古诗词的题目，为你生成一本诗集（epub 格式）
- [search-wiki](https://github.com/yeshimei/ntbl-scripts/tree/master/scripts/search-wiki) - 一个把文本词汇变成 markdown 添上搜索百度百科的链接和简介的脚本。
- [text-check](https://github.com/yeshimei/ntbl-scripts/tree/master/scripts/text-check) - 查询文本中的高频词。
